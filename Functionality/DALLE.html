Functionality

This sub-application can generate different photos by inputting some sentences from the text box. As the main page shows, DALLE will pass through the process of tokenization, embedding, etc. to get the photos that the user wants, but the user can choose a more suitable photo to do further generating



Function Procedure

Step 1: set the text input

User will input text or paragraph to the text box such as "Generate a photo of star night, anime style, peaceful," "Generate an anime character, angry, human" In here we will demonstrate "Generate a photo of cat" or user can input an artwork to regenerate

Figure 2: Users can input any conditions that the ChatGPT need to guarantee while generating the photo

Figure 3: User can choose which photo they like or regenerate again

Step 2: Output (or back to Tokenization depending on user choice)

After the process that told on the Main page, ChatGPT will output the picture with four choices

Step 2: Output the final product

Finally, user can choose their selected photo to download

Figure 4: User can choose which photo to download
